Beginner Boxes

Als je nieuw bent in de wereld van Hack The Box, zijn hier enkele van de eenvoudigere boxes die je kunt proberen om je vaardigheden op te bouwen.

Box 1: Jerry

Jerry is een zeer beginner-vriendelijke box die zich richt op basis webexploitatie. Je zult een webserver vinden die een kwetsbare Tomcat-installatie draait.

Box 2: Blue

Blue is een andere toegankelijke box waar je kunt leren over SMB-exploitatie en het gebruik van tools zoals Metasploit. Deze box is vooral nuttig voor het begrijpen van de kwetsbaarheden die werden misbruikt door de WannaCry ransomware.

Box 3: Nibbles

Nibbles is een box die je introduceert in basic Linux-exploitatie en privilege-escalatie. Je zult onder meer moeten werken met webshells en suid-bits.
