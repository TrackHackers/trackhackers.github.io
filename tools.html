Kali-Linux in WSL

Een van de meest handige manieren om Kali Linux te draaien op een Windows-systeem is via het Windows Subsystem for Linux (WSL). De installatie is eenvoudig; je activeert eerst WSL via de Windows-functies en installeert dan Kali Linux uit de Microsoft Store. Na de installatie heb je toegang tot een breed scala aan cybersecurity-tools direct vanuit je Windows-omgeving, zonder te hoeven dual-booten of een virtuele machine te draaien.

WSL met Kali Linux heeft mijn pentesting workflow drastisch verbeterd. Het is een game-changer! -- een trouwe lezer
Als DevSecOps engineer is het essentieel om flexibel te zijn. Met WSL kan ik nu naadloos schakelen tussen Windows en Linux. -- Fervente hacker

Hoe installeren?

Stap 1: WSL inschakelen Open PowerShell als beheerder. Voer het volgende commando uit om WSL te activeren: 

              
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
Stap 2: Virtual Machine Platform inschakelen Voer het volgende commando uit om het Virtual Machine Platform in te schakelen: 
              
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
Stap 3: Linux Kernel Update Package downloaden Download de nieuwste Linux Kernel van de officiële Microsoft-site. Voer het gedownloade MSI-pakket uit om de installatie te voltooien. Stap 4: WSL 2 instellen als standaard Open PowerShell opnieuw als beheerder. Voer het volgende commando uit: 
              
wsl --set-default-version 2
Stap 5: Kali Linux installeren Ga naar de Microsoft Store en zoek naar "Kali Linux". Klik op "Installeren" om het installatieproces te starten. Na de installatie, open Kali Linux en volg de instructies om het op te zetten. Stap 6: Kali Linux updaten en upgraden Open de Kali Linux terminal. Voer de volgende commando's uit: 
              
sudo apt update && sudo apt upgrade

OpenVPN

OpenVPN is een ander essentieel stuk gereedschap, vooral voor het verbinden met Hack The Box. HTB vereist een VPN-verbinding voor toegang tot hun labomgevingen. Nadat je je account hebt aangemaakt op Hack The Box, kun je een persoonlijk OpenVPN-configuratiebestand downloaden van hun website. Dit bestand kan je vervolgens gebruiken met de OpenVPN-client om een veilige verbinding tot stand te brengen. Voor Kali Linux en andere Linux-distributies kan dit meestal worden gedaan met een simpel commando zoals hieronder wordt weergegeven. 

          
sudo openvpn {jouw-configuratiebestand}.ovpn
